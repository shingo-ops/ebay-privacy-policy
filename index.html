Privacy Policy

Last updated: April 5, 2026

1. Overview

This application (eBay Listing Manager) is a Google Apps Script-based tool that accesses eBay APIs to manage product listings, policies, and inventory on behalf of authorized users.

2. Information Collection and Use

We do not collect, store, transmit, or share any personal information beyond what is strictly necessary for eBay API authentication and functionality.

What we access:

  • eBay API credentials (App ID, Cert ID, Dev ID) provided by you
  • eBay OAuth tokens for authentication
  • Product listing data you choose to manage
  • Business policies (shipping, payment, return policies)

3. Data Storage

All data is stored exclusively in your Google Spreadsheet. We do not operate any servers or databases that store your data.

Storage details:

  • eBay authentication tokens are stored in your Google Spreadsheet's "ツール設定" (Tool Settings) sheet
  • All product and policy data remains in your spreadsheet
  • No data is transmitted to third-party servers outside of Google and eBay

4. Data Security

Your data security is protected by:

  • Google's security infrastructure - All data is stored in Google Sheets with Google's enterprise-grade security
  • eBay's OAuth 2.0 protocol - Secure authentication without storing your eBay password
  • Automatic token refresh - Access tokens are refreshed automatically and expire after 2 hours

5. Third-Party Services

This application communicates with:

  • Google Apps Script - Execution environment
  • Google Sheets API - Data storage and retrieval
  • eBay API - Product listing and policy management

No other third-party services have access to your data.

6. Data Sharing

We do not sell, trade, or share your data with any third parties. Your data is only accessed by:

  • You (the spreadsheet owner)
  • Users you explicitly grant access to your Google Spreadsheet
  • eBay API (only for the purpose of managing your listings and policies)

7. Your Rights

You have complete control over your data:

  • Access - All data is visible in your Google Spreadsheet
  • Modification - You can edit or delete any data at any time
  • Deletion - You can delete the spreadsheet or remove the script to completely remove all data
  • Revocation - You can revoke eBay API access at any time through eBay Developer Portal

8. Code Review

The application code is managed using Google Apps Script. Access to the code is controlled by your administrator.

9. Changes to This Policy

We may update this privacy policy from time to time. The "Last updated" date at the top of this page indicates when the policy was last revised.

10. Contact Information

If you have any questions or concerns about this privacy policy or the application's data practices:

  • Review the application documentation provided by your administrator
  • Contact your system administrator or the person who provided you with this application
  • For technical issues, refer to the eBay Developer Portal documentation

This Privacy Policy is maintained for eBay OAuth authentication purposes.
